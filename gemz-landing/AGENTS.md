# AGENTS.md - Gemz Landing

## Project Overview
This is the landing page project for Gemz, built with Next.js. It is a separate Next.js application nested within the monorepo.

## Setup & Commands
- **Install:** `npm install`
- **Dev:** `npm run dev`
- **Build:** `npm run build`
- **Lint:** `npm run lint`

## Expectations
- **Framework:** Next.js 14+.
- **Styling:** Tailwind CSS.
- **Components:** Radix UI primitives, Shadcn UI.
- **Structure:** Standard Next.js App Router structure (`app/`, `components/`, `lib/`).
