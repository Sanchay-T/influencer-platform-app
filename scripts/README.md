# Scripts

- `analysis/`: one-off analysis helpers (keyword/run analyses).
- `ops/`: operational utilities (job/debug/rescue, run-test harnesses).
- `tools/`: small utilities (e.g., CSV conversion).
- `db-lint-migrations.js`: migration prefix linter (run via `npm run db:lint`).
- `legacy/`: archived helper scripts kept for reference; avoid adding new ones here.

Keep new scripts here instead of the repo root.
