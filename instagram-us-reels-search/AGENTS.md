# AGENTS.md - Instagram US Reels Search

## Project Overview
This module focuses on US-based Instagram Reels keyword search using Sonar Pro. It implements various search strategies including hybrid, AI-powered, and production-ready search flows.

## Setup & Commands
- **Install:** `npm install`
- **Search Commands:**
  - `npm run search`: Run Sonar Instagram search.
  - `npm run hybrid`: Run hybrid search.
  - `npm run final`: Run final search logic.
  - `npm run ai`: Run AI-powered search.
  - `npm run prod`: Run production search.
  - `npm run v2`: Run production search v2.

## Expectations
- **TypeScript:** Use `tsx` for execution.
- **Output:** Results are typically saved as JSON files (e.g., `ai-results-nutrition.json`).
- **Environment:** Requires `.env` with Sonar/Perplexity API keys.
